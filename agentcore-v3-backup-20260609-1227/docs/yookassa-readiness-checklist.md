## Что нужно сделать владельцу сайта

1. **SSL-сертификат**: Настроить nginx + Let's Encrypt на сервере 31.76.102.116
2. **Реальные реквизиты**: Заменить шаблонные данные на `/requisites`
3. **YooKassa env**: Заполнить `YOOKASSA_SHOP_ID` и `YOOKASSA_SECRET_KEY` в `.env`
4. **YooKassa SDK**: `npm install yookassa` на сервере
5. **Legal docs**: Проверить тексты с юристом (особенно `/terms` и `/privacy`)
6. **Тестовые платежи**: Пройти тестовые платежи через sandbox ЮKassa
7. **Модерация**: Подать заявку на подключение в личном кабинете ЮKassa
8. **Отзывы**: Заменить демо-отзывы на реальные (если есть) или убрать
