# AgentCore

Платформа управления AI-агентами для бизнеса. AgentCore позволяет создавать, настраивать и контролировать интеллектуальных агентов для автоматизации продаж, поддержки клиентов, исследований и других задач. Система предоставляет единый интерфейс для управления агентами, мониторинга их работы, биллинга и интеграций с внешними сервисами.

Проект построен на стеке **TanStack Start** (React + серверные функции), **Drizzle ORM** с **PostgreSQL** и локальной аутентификацией на базе JWT. Поддерживается модульная система подписок через **YooKassa**, детализированный аудит, версионирование конфигураций агентов и демо-режим для ознакомления.
